# Shaw — Human Entry Point

**What Shaw is**: Receiver-relative information value. Same data, different worth to different people. The formal framework splits Shaw into two independent components: Shaw-magnitude (how much changed — classical surprisal, intent-independent) and Shaw-resolution (what it means — intent-dependent). The difference between Shaw and Shannon is whether Intent is in the equation.

**Formal paper**: `ai/docs/shaw_hypothesis.tex` (2026 hypothesis paper — dual-component, QLM, falsifiable predictions)

**Terminology**: `ai/skills/txo-terminology.md` (Shaw section — loaded every session)

## Two Workflows, One Pipeline

```
Research (W-H-S) → Synthesis document → Adaptation → Audience documents
```

**Shaw Research** (Walk-Hammer-Synthesis): Find, test, lock understanding. The inner loop. Produces a Synthesis document — comprehensive, researcher-level.

**Shaw Audience** (Adaptation): Transform the Synthesis document for specific audiences. Each audience carries a different active Intent, producing different Shaw-resolution from the same source material. Five audience types: Boss (WHY), Lead (WHAT), Worker (HOW), User (USE), Student (BUILD codebook).
